# Universal Data Object (Observation Schema)

Every module in the LandSense AI platform communicates using the same **Observation** JSON structure. No module should invent its own JSON schema.

## Schema Details

```json
{
  "observation_id": "string (UUID)",
  "timestamp": "string (ISO 8601 UTC)",
  "latitude": 12.9716,
  "longitude": 77.5946,
  "images": [
    "string (Base64 encoded images or local file paths)"
  ],
  "voice_query": "string (Optional raw text of regional language voice query)",
  "construction_stage": "string (e.g., Excavation, Foundation, Structure, Finishing, Completed)",
  "confidence": 0.85,
  "progress": 45.0,
  "noise_db": 78.5,
  "dust_pm25": 42.0,
  "dust_pm10": 85.0,
  "sensor_status": "string (e.g., connected, disconnected, degraded)",
  "rera_projects": [
    {
      "name": "string (Project Name)",
      "builder": "string (Builder Name)",
      "status": "string (Approval Status)",
      "distance": 120.0
    }
  ],
  "development_score": 82.5,
  "summary": "string (AI PC LLM-fused summary reasoning about the development)",
  "embedding": [
    0.015, -0.024, 0.187
  ]
}
```

## Field Descriptions

| Field Name | Type | Description | Source |
| :--- | :--- | :--- | :--- |
| `observation_id` | String | Unique UUID for the scan. | Generated by OnePlus App / Backend |
| `timestamp` | String | ISO 8601 UTC formatted timestamp of when the scan occurred. | OnePlus App / Backend |
| `latitude` | Float | GPS Latitude of the scan site. | OnePlus App GPS |
| `longitude` | Float | GPS Longitude of the scan site. | OnePlus App GPS |
| `images` | Array | Base64-encoded strings of up to 4 images. | OnePlus App Camera |
| `voice_query` | String | Optional voice query translated to text or regional transcription. | OnePlus App Voice |
| `construction_stage` | String | Visual classification of the construction progress. | AI PC Inference |
| `confidence` | Float | Confidence level (0.0 to 1.0) of visual classification. | AI PC Inference |
| `progress` | Float | Progress completion percentage (0.0 to 100.0). | AI PC Inference |
| `noise_db` | Float | Average decibels measured during observation window. | Arduino IoT Sensor |
| `dust_pm25` | Float | PM2.5 environmental particulate level. | Arduino IoT Sensor |
| `dust_pm10` | Float | PM10 environmental particulate level. | Arduino IoT Sensor |
| `sensor_status` | String | Health status of IoT sensor node. | Backend / Adapter |
| `rera_projects` | Array | Nearby registered RERA projects matching location. | SQLite Local DB / MCP |
| `development_score` | Float | Explainable score (0-100) representing activity & compliance. | Fusion Engine |
| `summary` | String | Fused text overview summarizing RERA status and physical progress. | Fusion Engine |
| `embedding` | Array | Vector representation of visual features. | AI PC Inference |
